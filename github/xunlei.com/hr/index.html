企业文化

CULTURE

  • 以人为中心

    在迅雷，抵制「唯领导论」的观念，我们应着眼于身边的「每一位伙伴」，关注自我进步，也重视伙伴成长，与拼搏中的每个迅雷人一起，共同创造备受关怀的氛围。

  • 追求极致

    对自己的专业领域，怀抱极高的热情，相信我们的努力，可以让世界变得更加美好，时刻保持强大的自驱精神，贵在行动，绝不原地踏步，志在创见出更大的梦想。

  • 无惧挑战

    善于打破固有格局，不要安于现状，去尝试建立出新的游戏规则，为自己或团队的事业版图，开拓出新的疆界，并能为更多的用户创造出更大价值。

  • 善于突破

    面对改变，不胆怯不退缩，面对创业道路上的屡次试错，保持昂扬斗志，凡事先尝试，敢想象他人不敢想之事，善于发现新的契机点。

  • 简单开放

    拒绝繁文缛节，遵循「对事不对人」的原则，向人为制造低效率的大公司病say no，在迅雷，我们保持简单高效，倡导跨界沟通协作，让快乐的文化气氛无处不在。
