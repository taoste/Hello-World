# 宝云小站

> [宝云小站](https://baoyun.ren)

## 配置文件说明

配置项        | 含义
------------- | ----
`url`         | 首页链接
`title`       | 页面标题
`HTMLtitle`   | 在**页面**上显示的标题（`HTML`结构）
`name`        | 姓名
`birthInfo`   | 出生日期
`contact`     | 联系方式
`friendLinks` | 友情链接
`description` | 网页描述
`keywords`    | 关键词
`styles`      | 样式配置项
