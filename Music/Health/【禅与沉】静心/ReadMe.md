> [般若波罗蜜多心经](https://zh.wikipedia.org/wiki/般若波羅蜜多心經)【译文】

```
观自在菩萨，
行深般若波罗蜜多时，
照见五蕴皆空，
度一切苦厄。
舍利子，
色不异空，空不异色；
色即是空，空即是色。
受、想、行、识，亦复如是。
舍利子，是诸法空相，
不生不灭，不垢不净，不增不减，
是故空中无色，无受、想、行、识；
无眼、耳、鼻、舌、身、意；
无色、声、香、味、触、法；
无眼界，乃至无意识界；
无无明，亦无无明尽；
乃至无老死，亦无老死尽。
无苦、集、灭、道，无智亦无得。
以无所得故，菩提萨埵，
依般若波罗蜜多故，心无罣碍，
无罣碍故，无有恐怖，
远离颠倒梦想，究竟涅槃。
三世诸佛，依般若波罗蜜多故，
得阿耨多罗三藐三菩提。
故知般若波罗蜜多，
是大神咒，是大明咒，
是无上咒，是无等等咒，
能除一切苦，真实不虚。
故说般若波罗蜜多咒，
即说咒曰：
“揭谛、揭谛，波罗揭谛，
波罗僧揭谛，菩提萨婆诃。”
```
—————— 玄奘法师译，般若波罗蜜多心经
