## [Mixin](https://mixin.one/) [Messenger](https://mixin.one/messenger) 新手指南

欢迎使用 Mixin Messenger，尽享端对端加密聊天和全币种多链钱包：

- 安全：去中心化钱包
- 快捷：无需等待 1 秒到账
- 免费：转账不收手续费
- 易用：6 位数字密码轻松管理资产，通讯录转账不再转错人
- 丰富：支持 BTC、EOS、ETH、XMR 等 28 条主链，超过 10 万种代币
