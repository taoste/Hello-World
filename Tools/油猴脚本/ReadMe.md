```
如果捕获列表内出现的链接以Blob开头，则需要等待音乐或视频加载完闭，再点击下载，才能下载到完整的音乐或视频。

Firefox中指向Blob URL的链接元素不允许download属性，所以脚本在Firefox下对MediaSource流式播放器的支持不完整，会出现无法下载之类的问题。
```
