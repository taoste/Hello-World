> 事实上，包括牙膏、洗面乳…等一些「软管」包装的产品，在包装底部都标有各种颜色的色条，该色条的正确名称为「**电眼定位点**」，是软管产品在生产过程中，提供机器对产品「**封尾**」时的定位与识别，当产品装填入软管后，封尾机器上的电眼会感应色条，对软管进行封管及裁剪的包装作业。而色条选色的基本概念，以与底色反差较大并且容易反光为主，方便电眼进行感应，也因为黑色条标的感应效果最好，所以也最常见。
