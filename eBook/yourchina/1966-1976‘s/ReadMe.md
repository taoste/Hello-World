>> 无产阶级文化大革命，通称文化大革命、简称文革，是中华人民共和国历史上的一场政治运动，于1966年5月16日—1976年10月6日间发生在中国大陆境内。因其时间长达十年之久，故也被后世称为“十年动乱”或“十年浩劫”。<br>文革是由时任中国共产党中央委员会主席的毛泽东与中央文化革命小组，自上而下动员成千上万红卫兵在中国大陆进行全方位发动的阶级斗争。
